# ANPR-Fine-Generation-System
A real-time ANPR system using YOLOv8, EasyOCR, Flask, and SQLite for vehicle plate detection and text extraction. Detects plates from live/video feeds, recognizes characters, and stores results with timestamps.

✨ Features
Real-time license plate detection using YOLOv8
Accurate OCR text extraction with EasyOCR
Supports live camera, images, and video input
Stores plate numbers with date & time logs
User-friendly Flask web interface
Lightweight SQLite database integration
